Joomla 3.0 is beschikbaar! Dit is de eerste groote release sinds Joomla 2.5. Deze versie is de eerste in de reeks voor de lange termijn ondersteuning binnen Joomla. De lange termijn versies zijn bijv. Joomla 2.5, Joomla 3.5 en Joomla 4.5. Alle tussenliggende versies zijn bedoeld om nieuwe functionaliteiten te ontwikkelen. Deze tussen versies kunnen eventueel gebruikt worden voor productie doeleinden maar hebben dus geen lange termijn ondersteuning van Joomla, kortom upgraden is belangrijk!

Moet ik nu Joomla 2.5 upgrade?

Joomla adviseert alle gebruikers om een upgrade naar Joomla 3.0 voor te bereiden. Met een Joomla 2.5.x versie hoef je nu niet direct te upgraden. Aangezien Joomla 3.0 net is uitgekomen zijn nog niet alle componenten, modules, plugins en templates aangepast op de nieuwe functionaliteiten van Joomla 3.0. Maar dit zal niet lang duren aangezien de overgang van Joomla 2.5 naar Joomla 3.0 relatief klein is. Het is dus enkel wachten op de ondersteuning van de leveranciers. In de Joomla Extentie Directorie (JED) wordt al duidelijk aangegeven welke componenten, modules, plugins en templates al werken met Joomla 3.0.

Wat betekent dit nu voor Joomla 1.5?

Met de komst van Joomla 3.0 is nu ook het einde van Joomla 1.5 in zicht. Binnenkort zal er geen support meer worden geleverd op deze versie. Er worden dan op versie 1.5 geen security patches meer aangeboden. Hierdoor loopt een website die draait op Joomla 1.5 een steeds groter risico om gehacked te worden. Joomla 1.5 kun je nog wel gebruiken maar het is aan te raden om te upgraden naar minimaal Joomla 2.5 vooral om de beschikbaarheid van de website te kunnen garanderen.

Wat is er nieuw in Joomla 3.0?

  • Incorporation of Twitter Bootstrap into a jui media package.
  • A new responsive, mobile device friendly administrator template--Isis-- and interface.
  • A new front end template--Protostar-- built using Twitter Bootstrap.
  • Updated accessible template called Beez3.
  • A simplified 3-step installation process.
  • PostgreSQL Driver. You will be able to run Joomla 3.0 sites using the PostgreSQL database.
  • PHP Memcached Driver
  • Use of JFeed for feed management rather than SimplePie
  • Installation of language packages directly from the extension manager
  • Guest user group present by default
  • Saving blank articles allowed
  • New administrator statistics module
  • Update TinyMCE to version 3.5.6
  • Continued clean up of older unused code, files and database fields and tables.
  • Improvements to Smart Search
  • Extensive work on code style standardisation and consistency
  • Unit testing in the CMS
  • Updated system tests in the CMS
  • Custom active menu item for menu module
  • Let the SEF plug-in add the canonical url to the head
  • Version 12.2 of the Joomla Platform

Joomla 3.0 bevat:

  • 34 nieuwe functies (inclusief 15 functies van Joomla 2.5.1 - 2.5.7)
  • Geen vast veiligheidsproblemen
  • 491 gemelde problemen
