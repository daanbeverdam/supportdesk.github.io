Joomla heeft onlangs versie 2.5.5 (op dinsdag 19 juni 2012) een Security Release uitgebracht, maar hier bleken nog wat schoonheidsfoutjes in te zitten. Ondanks dat er verschillende bugs in zaten werd het diezelfde dag nog opgelost door de community middels een nieuwe stabiele 2.5.6 versie.

Even in het kort gezegd wat deze Joomla 2.5.5 release inhoud:

  • 2 beveiligingsupdates (Medium High, Low)
  • 11 nieuwe mogelijkheden
  • 113 bugs opgelost

In versie Joomla 2.5.6 is deze Security Release verder gestabiliseerd.
Lees meer over deze release op de Joomla website.
